    使用说明

    基本流程

    1. 顶栏下方选 闭合 / 附合 模式
    2. 填入起点坐标、起始方位角（° ′ ″ 三栏，或点「十进制」直接输小数度）
    3. 附合模式再填终点坐标 + 终止方位角
    4. 测站角度表填各测站观测角（点号 + ° ′ ″），边长表填各条边的水平距离
    5. 点底部 「🚀 计算」 按钮 → 下方出现结果表、闭合差、略图
    6. 改任何输入后，底部按钮会变橙提示「已修改 · 点此重算」

    起算方位角输入方式

    • 度分秒（默认）：° ′ ″ 三栏，秒支持小数
    • 十进制：点「⇄ 十进制」切换，单个输入框直接输小数度（如 96.86）
    • 反算：点「📐 反算」展开，填入另一个已知点 B 的坐标，自动用 atan2 算出 α_AB（X=北, Y=东 约定下：α = atan2(ΔY, ΔX)）。附合模式终边同理：填 C→D 反算 α_CD

    角度类型

    • 左角：前进方向左侧的夹角 → 沿导线逆时针行进时使用
    • 右角：前进方向右侧的夹角 → 沿导线顺时针行进时使用

    测站表 vs 边长表

    • 测站角度表：每个测站一行（点号 + ° ′ ″），不含边长
    • 边长表：每条边一行（起点 → 终点 + 边长），独立于点号
    • 闭合导线：第 n 行标签自动是 "最后一个点 → 起点"（闭合边）
    • 附合导线：第 n 行标签自动是 "最后一个点 → 终点"（终止边）
    • 改「测站角度表」里的点号，边长表的标签会自动跟着变

    闭合差

    • fβ 角度闭合差，限差 ±40″·√n
    • K = fs / ΣD 全长相对闭合差，限差可调（默认 1/2000）
    • 超限时不阻断，仍给出平差结果并在顶部显示警告

    整数修正模式

    「限差/角型」卡最下方的复选框。勾上后 v_β 取整秒、vx/vy 取 1mm 整数倍，Σ 严格等于 -fβ / -fx / -fy，余量按 |β| / 边长 largest-remainder 分。符合测量手工报表习惯。

    • v_β：整数模式显示整秒（+7 / -6）；小数模式显示 1 位小数（+6.3）
    • vx / vy：整数模式 3 位小数（1mm）；小数模式 4 位小数
    • 例：fβ=-25″，4 角 → vβ=[+7, +6, +6, +6]（最大 |β| 拿 +1 秒）

    结果表列结构（12 列）

    角度组：观测角 ｜ v_β ｜ 改正后 ｜ 方位角

    边长 / 改正数 / 坐标：边长 ｜ X' ｜ Y' ｜ vx ｜ vy ｜ ΔX ｜ ΔY ｜ X ｜ Y

    • v_β：角度改正数（带符号，整数模式整秒 / 小数模式 1 位）
    • X' / Y'：未修正的坐标增量（带撇 = 调整前）
    • vx / vy：坐标增量改正数
    • ΔX / ΔY：修正后的坐标增量
    • X / Y：调整后最终坐标

    导出

    • TSV：制表符分隔，复制后直接粘到 Excel 单元格自动对齐
    • PNG：结果表截图，可发微信/邮件
    • JSON：完整工程备份

    数据保存

    所有数据自动存到本地浏览器，不会上传服务器。点击顶部 💾 命名保存方案，📂 切换历史方案。

    添加到主屏幕

    手机浏览器打开后，可在菜单选择"添加到主屏幕"，即可像 App 一样使用。
